# 🌟 KeenKeeper

> Stay connected with the people who matter most in our lives ❤️

---

## 📖 Description

**KeenKeeper** is a modern web application designed to keep friends and family connected in today’s busy world.  
It helps you maintain meaningful relationships through simple, intuitive interactions and thoughtful features.

---

## 🚀 Tech Stack

- ⚛️ React
- 🎨 Tailwind CSS
- 🌼 DaisyUI
- 🔥 React Icons

---

## ✨ Features

- 💬 Real-time communication (chat / updates)
- 👨‍👩‍👧‍👦 Manage friends & family groups
- 🔔 Smart reminders to stay in touch
- 🎨 Clean and responsive UI
- 🌙 Dark mode support
- 📱 Mobile-friendly design
